# -
مشروع موقع “الحضارة الإسلامية” هو موقع إلكتروني تعليمي وثقافي يهدف إلى التعريف بتاريخ الحضارة الإسلامية وإنجازاتها في مختلف المجالات مثل العلوم، الطب، الفلك، العمارة، الأدب، والفنون. يوفر الموقع معلومات موثوقة ومنظمة بطريقة سهلة تساعد الزوار على استكشاف تأثير الحضارة الإسلامية في تطور العالم عبر العصور.
